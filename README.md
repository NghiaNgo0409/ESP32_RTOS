# ESP32_RTOS

### Device
- Yolo UNO (ESP32-S3)
- LCD I2C
- AIOT kit:
  - Temperature & Humidity Sensor
  - Soil Moisture Sensor
  - Relay, Light Sensor
  - LED

 ***
 ### Tasks
 - Display value getting from sensors on LCD I2C
 - Toggle relay when detect partical soil moisture values
 - Toggle LED when detect partical light sensor values
